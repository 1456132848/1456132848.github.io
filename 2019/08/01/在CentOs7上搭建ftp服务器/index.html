在CentOs7上搭建ftp服务器

发表于 2019-08-01

一、安装 VSFTPD

yum install -y vsftpd

二、启动 VSFTPD

service vsftpd start

启动后，可以看到系统已经监听了 21 端口：

ps -ef | grep 21

此时，访问 ftp://127.0.0.1 可以浏览ftp目录

三、创建 FTP 用户

创建用户

useradd ftpuser

设置密码

echo "javen205" | passwd ftpuser --stdin

四、 远程访问ftp

关闭SELinux服务：

setenforce 0

关闭防火墙：

iptables -F

然后可以在远程使用连接命令：

ftp 10.211.55.4(ip地址)

总结

以上步骤采用的均是ftp默认配置，若需要修改配置，自行搜索。
